One Column Template

Text Styles

This page shows all of the available styles for the public facing Central College template. Do not use inline styles to side-step the layout.

Grid

This template uses a twelve column grid system. There are additional helper classes for one-third, two-thirds and one-half columns. The grid containers should always add up to twelve. It is also helpful to follow the containers with a .clearBoth div.

Icons

This template has access to the Font Awesome icon library. This is the preferred method for using icons including social network options like , or .

Headings

Heading 1 is reserved for section titles at the top of the template. Heading 2 is typically used for page titles. Use Heading 3 to break up copy with appropriate headings. Heading 4 should be used to divide copy within a heading. Heading 5 should be used sparingly.

Buttons

There are two button styles available. The primary button should be used to indicate to the user what the intended default action is. The secondary button is for auxillary featues (i.e. resetting a form.). Button styles will form on form controls or with anchor tags.
